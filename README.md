# 🌀 CZ: C Lang but Make It Gen Z

> 🚀 A chaotic fusion of C programming and Gen Z lingo, brought to life with Flex & Yacc. It’s giving programming vibes.

## 👀 What Even Is CZ?

CZ (C-Zee or C-Zed, idk and idc) is a *spicy remix* of the C programming language that speaks your language — like literally. Think `yeet` instead of `return`, `vibing` instead of `for`, etc.

Built using:
- 💪 **Flex** – for tokenizing all that slang.
- 🧠 **Yacc/Bison** – for making sense of the Gen Z logic.

## 🛠️ Build & Run It, Baddie

Make sure you’ve got:
- `flex`
- `bison` (or `yacc`)
- `gcc`

Building & Running:

```bash
make build-run
```
Input your Gen Z program bestie 🔥
